# ZombieEscape Player Wiki

Welcome to the player guide for **ZombieEscape**, an open-world, location-based zombie survival game for Android. Real streets and buildings become the game world: explore, loot, fight, trade, build a home, recruit survivors, and keep yourself alive.

> ZombieEscape is in active development. Screens, balance, and mechanics may change. This wiki describes game version **1.1.9**.

## The core survival loop

**Move → search → loot → manage supplies → improve equipment → survive combat → return home.**

Your world, inventory, equipment, injuries, vehicles, NPCs, houses, interiors, and looted buildings are saved persistently.
